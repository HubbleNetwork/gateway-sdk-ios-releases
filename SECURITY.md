# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the Hubble Gateway SDK, please report it privately by emailing:

**security@hubble.com**

Please include:

- A description of the vulnerability
- Steps to reproduce
- The affected version (e.g. `0.3.0`)
- Any potential impact

We'll acknowledge your report within 2 business days and aim to provide a status update within 7 days.

Please do **not** open a public GitHub issue for security vulnerabilities. We'll coordinate disclosure with you once a fix is available.
